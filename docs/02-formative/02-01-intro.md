# Форматив


Текст в новом ифкуиле, так же как и тексты на естественных языках, состоит из предложений, предложения складываются из слов, слова из фонем.

В новом ифкуиле есть три части речи, три типа слов:

1. **Форматив**. Эта часть речи несет основную смысловую нагрузку; предложения, состоящие из одних только формативов - не редкость. Они выполняют роль существительных, глаголов, прилагательных, наречий и многих других частей речи естественных языков.
2. **Референтив**. Аналог местоимений из естественных языков, снабженный расширенными возможностями.
3. **Адъюнкт**. Разнородная группа вспомогательных слов, выполняющих различные задачи, например: вынос морфологической категории за пределы форматива, маркировка прямой речи, междометия.

В настоящем разделе рассматриваются формативы, другие части речи будут рассмотрены в отдельных разделах.
