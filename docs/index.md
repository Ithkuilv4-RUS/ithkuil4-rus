# Новый Ифкуиль

На этом сайте представлена грамматика Нового Ифкуиля, проект сконструированного языка, пригодного для использования человеком.

Язык использует матрицу грамматических концептов, предназначенную для более явного, логичного и точного выражения более глубоких уровней человеческого мышления по сравнению с естественными языками.

Язык основан на предшественнике, Ифкуиле, но был значительно изменен с учетом различных предложений по улучшению от поклонников языка-предшественника (см. [Приложение C](https://www.ithkuil.net/newithkuil_appendices.htm#Appendix_C) со списком участников), а также новые идеи автора.

По сравнению с предшественником, Новый Ифкуиль гораздо более систематизирован и упорядочен в своей структуре, что потенциально упрощает его изучение.
