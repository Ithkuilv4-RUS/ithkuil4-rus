# Фонология
Фонология языка описывает его систему произносимых голосом звуков и то, как эти звуки выстраиваются в последовательность для составления слов.

Фонемы языка это значимые произносимые голосом звуки, используемые для образования слов. Каждый такой звук может иметь одну или несколько вариаций (называемых аллофонами) в зависимости от его фонетического окружения (т.е. какие другие звуки примыкают к нему).

Поскольку нативная система письма языка Ифкуиль не является полностью фонетической, для обозначения фонем этого языка используются буквы латинского алфавита с диакритическими знаками. При объяснении правил произношения фонем  в этой главе, также испозованы обозначения Международного фонетического алфавита (IPA).
